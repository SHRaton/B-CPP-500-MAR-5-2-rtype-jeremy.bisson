Description détaillée

Gère la logique de jeu côté serveur.

Cette classe gère toute la logique de jeu côté serveur, incluant la gestion des timers, des collisions, du spawn des monstres et des power-ups. Elle utilise Lua pour la configuration et la personnalisation du gameplay.
