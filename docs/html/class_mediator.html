Description détaillée

Médiateur pour la communication entre le réseau et la logique de jeu.

Cette classe implémente le pattern Mediator pour faciliter la communication entre les composants réseau et la logique de jeu. Elle permet de découpler ces composants tout en assurant leur synchronisation.
